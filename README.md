# Used Car Sales Analysis

This project focuses on analyzing used car sales data using two different data sources:
1. A publicly available Kaggle dataset
2. Web-scraped used car listings from the Droom website

## Project Structure
- kaggle_used_car_data_analysis/
  - Contains analysis performed on the Kaggle used car dataset
- droom_web_data_analysis/
  - Contains web scraping and analysis of used car data from Droom

## Tools & Technologies
- Python
- Pandas
- NumPy
- BeautifulSoup
- Jupyter Notebook

## Work Done
- Collected used car data from Kaggle
- Performed web scraping on Droom website
- Cleaned and prepared datasets
- Conducted exploratory data analysis

## Outcome
The project helps in understanding pricing trends and important factors
that influence used car prices in the market.
